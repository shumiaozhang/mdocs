# 正则规则

### 1. 身份证号

```js
/^[1-9]\d{7}((0\d)|(1[0-2]))(([0|1|2]\d)|3[0-1])\d{3}$|^[1-9]\d{5}[1-9]\d{3}((0\d)|(1[0-2]))(([0|1|2]\d)|3[0-1])\d{3}([0-9]|X)$/
```
```js
/(^\d{15}$)|(^\d{18}$)|(^\d{17}(\d|X|x)$)/
```
### 2. 手机号

```js
/1[0-9]{10}/
```
### 3. 金额(保留两位小数)

```js
/(^[1-9]\d*(\.\d{1,2})?$)|(^0(\.\d{1,2})?$)/
```
### 4. 年龄

```js
/^(?:[1-9][0-9]?|1[01][0-9]|120)$/
```
### 5. 温度

```js
/^(((3[5-9]|4[0-3])(\.\d)?)|44)$/
```
