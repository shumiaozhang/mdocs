### 1. 进度条

```css
* {
  margin: 0;
  padding: 0;
  list-style: none;
}
.header_out {
  padding: 20px;
  display: flex;
  text-align: center;
}
.setps_item {
  flex: 1;
}
.setps_item_out {
  display: flex;
  justify-content: center;
  height: 20px;
  position: relative;
}
.setps_title {
  width: 20px;
  height: 20px;
  border-radius: 50%;
  color: #fff;
  background: #ddd;
  text-align: center;
  line-height: 20px;
  display: flex;
  justify-content: center;
  z-index: 3;
}
.setps_line {
  height: 2px;
  width: 100%;
  position: absolute;
  z-index: 2;
  top: 50%;
  background: #ddd;
}
.setps_content {
  color: #ddd;
  font-size: 10px;
  margin-top: 10px;
}
.start {
  right: 0;
  width: calc(50% - 10px);
}
.end {
  left: 0;
  width: calc(50% - 10px);
}
.active-color {
  color: skyblue;
}
.active-bg {
  background: skyblue;
}
```
```html
<div id="app">
  <div class="header_out">
    <div class="setps_item" v-for="(item, index) in setps" :key="index">
      <div class="setps_item_out">
        <div class="setps_title" :class="[value >= (index + 1) ? 'active-color' : '']">{{index + 1}}</div>
        <div class="setps_line" :class="[index === 0 ? 'start' : 0, index === setps.length -1 ? 'end' : 0, value >= (index + 1) ? 'active-bg' : '']"></div>
      </div>
      <div class="setps_content" :class="[value >= (index + 1) ? 'active-color' : '']">{{item}}</div>
    </div>
  </div>
</div>
```
```js
new Vue({
  el: '#app',
  data: {
    setps: ['a', 'b', 'c', 'd'],
    msg: 1,
    value: 2
  },
  mounted() {
    console.log(this.msg);
  },
  methodes: {}
})
```
![例子1](/images/例子11.png)

通过传入的`value`可改变进度条进行到的位置。

注：这里用到了一个css函数`calc()`, 用于计算长度，右侧和左侧的宽度一半用于这个函数。
