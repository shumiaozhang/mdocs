第一种：就是在参数列表前加上new命令。

```typescript
type AnimalConstructor = new () => Animal;

```

第二种：采用对象形式

```typescript
type F = {
  new (s:string): object;
};
```

某些函数既是构造函数，又可以当作普通函数使用，比如`Date()`。这时，类型声明可以写成下面这样。

```typescript
type F = {
  new (s:string): object;
  (n?:number): number;
}

```

上面示例中，F 既可以当作普通函数执行，也可以当作构造函数使用。
